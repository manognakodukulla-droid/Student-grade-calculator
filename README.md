# Student Grade Calculator

A web-based Student Grade Calculator built using **HTML, CSS, and JavaScript**.  
This application allows users to enter marks for multiple subjects, calculates the total and average scores, and assigns grades based on predefined criteria.

## Features
- Input marks for multiple subjects  
- Calculates total and average scores  
- Assigns grades automatically based on grading rules  
- Displays results in real time  
- Simple and responsive user interface  

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter marks for each subject in the input fields.
2. Click the **Calculate** button.
3. View the total, average, and grade instantly.

## Project Purpose
This project was created to practice frontend web development concepts and demonstrate the use of JavaScript for logic implementation and DOM manipulation.
